# Vault

> **Internal Reference:** This is [Stash](https://stashapp.cc/) - an organizer for adult media.

## Purpose
Media metadata management and scene organization with support for:
- Automatic scene detection and tagging
- Performer/studio metadata
- Integration with Whisparr (phantom) for library management

## Access
- **Local:** https://vault.homelab
- **Tailscale:** https://vault (via Tailscale network)

## Integration
- Whisparr connects to this service for metadata enrichment
- API available at `http://vault.siloarr.svc.cluster.local:9999`
